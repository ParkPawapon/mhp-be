# Security Policy

## Supported Versions

We support the latest release on the `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security reports.

Preferred: Use GitHub Security Advisories for this repository.
If Security Advisories are not available, contact the repository owner via
GitHub with details and a private reproduction.

When reporting, include:
- Affected version(s)
- Steps to reproduce
- Impact and severity assessment
- Any possible mitigations or workarounds

We aim to acknowledge reports within 3 business days.
